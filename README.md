# Trail Atlas

A React + TypeScript web app that explores U.S. National Parks using the National Park Service (NPS) REST API.

Built as a portfolio project to demonstrate modern frontend architecture, API integration, and scalable state management.

---

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router
- TanStack Query
- Axios
- Tailwind CSS
- ESLint + Prettier

---

## Features (Planned / In Progress)

### Phase 1 (Complete)
- App shell and layout system
- Dark mode! (Gotta have it)
- Routing structure:
  - Home
  - Parks
  - Favorites
  - Trip Planner (planned)
  - About
- Design system with Tailwind tokens
- API-ready architecture (no API calls yet)
- Theme persistence via localStorage

### Phase 2 (In Progress)
- NPS API integration
- Parks listing with filters
- Park detail pages
- Search + pagination
- React Query caching layer

---

This project uses the National Park Service API.
